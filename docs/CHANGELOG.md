# Changelog

Wszystkie istotne zmiany w projekcie SmartQuote-AI będą dokumentowane w tym pliku.

Format bazuje na [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), a projekt stosuje się do [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-30

### Added
- Inicjalizacja projektu SmartQuote-AI (Next.js 16).
- System autentykacji (NextAuth.js).
- Dashboard z podsumowaniem statystyk.
- Moduł Zarządzania Klientami (CRUD).
- Moduł Zarządzania Ofertami (CRUD, generator PDF).
- Moduł Zarządzania Kontraktami.
- Moduł Follow-upów z systemem przypomnień.
- Integracja z AI (Chat, generowanie ofert, analiza klientów).
- Panel ustawień profilu, firmy i kluczy API.
- Globalny Chat AI dostępny z poziomu dashboardu.
- Responsywny interfejs użytkownika z wykorzystaniem Tailwind CSS i Lucide React.
